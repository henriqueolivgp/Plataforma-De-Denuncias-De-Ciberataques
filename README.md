### Front-end Web

Clone the project

```bash
  git clone [https://link-para-o-projeto](https://github.com/henriqueolivgp/Plataforma-De-Denuncias-De-Ciberataques.git)
```

Enter the project directory


```bash
  npm i
```

```bash
  npm run dev
```
